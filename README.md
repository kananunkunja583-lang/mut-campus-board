# 🌿 CampusConnect

CampusConnect is a university community web platform built using HTML, CSS, and JavaScript.

## Features

* 📢 Announcements & Events
* 🔥 Anonymous Gossip & Confessions
* 🌿 Student Business Marketing
* 🔐 Admin Approval System
* 💾 Local Storage Persistence
* 🛡 Basic Profanity Filter
* 📱 Responsive Design

## Project Structure

* HTML Pages
* Global CSS Styling
* Modular JavaScript Files
* Admin Moderation Panel
